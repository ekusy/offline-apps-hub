# ダンジョンクローラー仕様書

## 1. 概要
7×7 のグリッドダンジョンを探索するターンベース RPG。階層を進みながら敵を倒し、アイテムを集めて到達階を更新する。

## 2. 機能要件

### 2.1 ダンジョン
- 7×7 グリッド × 10 階
- 各階に階段（▼）、敵 2〜5、アイテム 1〜3、未探索マスは霧で隠す（視界はチェビシェフ距離 ≤ 2）

### 2.2 プレイヤーステータス
- HP / MP / 攻撃 / 防御 / Lv / EXP
- 初期: HP 30 / MP 10 / 攻撃 5 / 防御 2

### 2.3 ターン
- プレイヤーの行動を選択 → 敵が応答
- プレイヤー行動:
  - 移動（上下左右 1 マス）
  - 攻撃（隣接敵）
  - 魔法（MP 4 消費、隣接敵に 1.5x ダメージ or 全画面雷で全体ダメージなど 1 種類）
  - 防御（次ターン受けるダメージ半減）
  - 待機（次の敵を見る）

### 2.4 敵
- 種別: スライム / ゴブリン / オーガ / ボス
- 隣接時に攻撃、それ以外はプレイヤーへ 1 マス接近
- HP/攻撃/防御は階層と種別でスケール

### 2.5 アイテム
- 回復ポーション（HP+15）
- MP ポーション（MP+5）
- 武器/防具（攻撃 +N / 防御 +N、上書きでアップグレード）

### 2.6 進行
- 階段に乗って次階へ
- 10 階クリアでエンディング
- 死亡時は階数とスコアを保存して終了

### 2.7 セーブ
- 入力ごとに localStorage に状態保存
- リロードで続きから再開

### 2.8 リーダーボード（ローカル）
- 最高到達階
- 最大ダメージ
- 最短クリアターン

## 3. 非機能要件
- ピュア HTML/CSS/JS（Canvas 不要、DOM ベース）
- オフライン動作
- 日英対応
- モバイル/PC 対応（タッチ + キーボード対応）

## 4. UI
- 中央: ダンジョン 7×7
- 上部: HP/MP/Lv/階/EXP/装備
- 下部: 行動ボタン（攻撃/魔法/防御/待機）
- 移動: 上下左右ボタン or 矢印キー / WASD
- ログエリア: 最新 4 行

## 5. ストレージキー
- `dungeon-state`: 現在ゲーム
- `dungeon-best-floor`: 最高到達階
- `dungeon-best-damage`: 最大ダメージ
- `dungeon-best-turns`: クリア時最短ターン

## 6. アクセスフロー
- `/apps/dungeon/` に配置
- `apps.js` 登録（カテゴリ: games / サブカテゴリ: rpg）
