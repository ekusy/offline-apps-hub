# 数独（Sudoku）仕様書

## 1. 概要
9×9 のマスを 1〜9 の数字で埋めるクラシックパズル。本プロジェクトのオフラインアプリハブに統合し、`/apps/sudoku/` に配置する。

## 2. 機能要件

### 2.1 ゲームルール
- 9×9 グリッドを 1〜9 で埋める
- 各行・各列・各 3×3 ブロックに 1〜9 がそれぞれ 1 個ずつ
- 初期配置（ヒント）は固定で編集不可

### 2.2 難易度
- やさしい / ふつう / むずかしい（ヒント数の多寡で表現）
  - やさしい: 40〜45 ヒント
  - ふつう: 32〜36 ヒント
  - むずかしい: 26〜30 ヒント

### 2.3 操作
- マスをタップ → 選択中（ハイライト）
- 画面下部の数字パッド（1〜9 / 消去）で入力
- メモモード（鉛筆）切替トグル：候補数字を小さく表示
- 元に戻す（最大 50 手）

### 2.4 補助機能
- ミス検知（オプション）：行/列/ブロック内重複をリアルタイム赤表示
- ハイライト：同じ数字のマスを薄く強調
- 進捗保存：localStorage に状態を保存し、リロードで継続可能
- タイマー、手数（入力回数）、ベストタイムを表示

### 2.5 終了条件
- すべて埋まり、ルール違反がなければクリア
- クリアダイアログでタイム表示・新しいゲーム提供

## 3. 非機能要件
- ピュア HTML/CSS/JS、フレームワーク不使用
- オフライン動作（Service Worker / Cache API 経由）
- 日英対応（i18n.js を利用）
- モバイル/PC 両対応のレスポンシブ UI

## 4. 画面構成
1. ヘッダー: タイトル + 言語切替 + ホームへ戻る
2. ツールバー: 難易度選択 / 新ゲーム / 元に戻す / メモ / ミス表示切替
3. ステータス: 手数 / タイム / ベスト
4. 9×9 ボード
5. 数字パッド（1〜9 + 消去）
6. ダウンロードボタン

## 5. ストレージキー
- `sudoku-difficulty`: 直近選択した難易度
- `sudoku-best-easy/normal/hard`: 各難易度のベストタイム（秒）
- `sudoku-state`: 現在のゲーム状態（任意保存）

## 6. URL パラメータ
- `?difficulty=easy|normal|hard`: 難易度直接指定
- `?seed=<uint32>`: パズルシード共有

## 7. アクセスフロー
- ハブからカード経由で `/apps/sudoku/` に遷移
- `apps.js` に登録（カテゴリ: games / サブカテゴリ: puzzle）
