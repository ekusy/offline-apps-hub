# カードバトル仕様書

## 1. 概要
プレイヤー対 AI の 1 対 1 カードバトル。8 枚のデッキを構築し、ステージを進めていく。

## 2. 機能要件

### 2.1 ゲームルール
- 双方 HP 20 から開始
- 毎ターンエネルギー +1（上限 10、初期 1）
- ターン開始時 1 枚ドロー（手札上限 5）
- カードはコスト（1〜4）と効果を持つ
- HP 0 で敗北、相手 HP 0 で勝利

### 2.2 カード効果
- 攻撃 +N: 敵に N ダメージ
- 盾 +N: 次ターン受けるダメージを N 軽減（1 ターン）
- 回復 +N: 自分 HP +N
- 強化 +N: 次に出す攻撃カード +N ダメージ
- 連撃: 1 コストで 3 ダメージ x 2 回

### 2.3 デッキ編集
- 全カードプール（20 種）から 8 枚選択
- localStorage に保存
- 初期デッキはプリセット

### 2.4 ステージ
- 全 5 ステージ。敵 HP/AI 強さが上昇
- ステージクリアでアンロックカードを獲得
- 全クリアで通常エンディング

### 2.5 敵 AI
- ランダム + 簡易ヒューリスティック:
  - HP < 30% なら回復優先
  - 強化があれば溜めてから攻撃
  - その他はランダム

### 2.6 統計
- 連勝数 / 最速クリアターン / 累計勝利数

## 3. 非機能要件
- HTML/CSS/JS のみ
- オフライン動作
- 日英対応
- モバイル/PC 対応

## 4. 画面構成
- 上部: 敵 HP、敵手札（裏面数）
- 中央: ボードログ（攻撃エフェクト/数値）
- 下部: 自分の HP、エネルギー、手札（5 枚）、デッキ残数、捨て札数
- 操作: 手札タップ → プレイ
- ターン終了ボタン

## 5. ストレージキー
- `cb-deck`: 選択デッキ（カード id 8 件）
- `cb-unlocks`: アンロック済みカード id
- `cb-stage`: 現在のステージ番号
- `cb-best-streak`: 連勝記録
- `cb-best-turns`: 最速クリアターン

## 6. アクセスフロー
- `/apps/card-battle/`
- `apps.js` 登録（カテゴリ: games / サブカテゴリ: cards）
